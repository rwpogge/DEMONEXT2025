# DEMONEXT Documents

This repository contains supplementary documents for the primary DEMONEXT system components. 
This ensures that we have the documents for the as-built purchased components, as many
will likely continue to evolve or be discontinued, and vendors may not always have copies of the
documentation for what we have in the future. It also provides a one-stop source for documents
potentially useful for field service of the as-built DEMONEXT system.
